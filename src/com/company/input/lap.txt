                  THE "LAP"--WHAT IT IS, WHAT'S IN IT
Hello, and thank you for downloading the LAP - List of Arcade Ports.
For those of you who have become hopelessly addicted to one or many arcade
games over the years and have wondered if you could possibly find a home
version and start saving your quarters, wonder no more.  The LAP lists the
arcade games and their derivatives that have been ported to the major home
systems, both game and computer.  For the sake of size, some information about
these ports, including the company name, is left out.

Please note that some of these games were only regionally released (say,
Japan, Europe, or America only), and finding them elsewhere may be
difficult.  Also, some ports are public domain or shareware.  I claim
no responsibility over the legality of these ports.

Additions?  Corrections?  Please e-mail them to me, at gt6005a@prism.gatech.edu
If you have any questions about a particular game or port, I would
first suggest trying the newsgroups and FAQs dedicated to those systems
before asking me.  My knowledge on some of these games is very minute, and
I've relied on the gracious expertise of others to make the LAP as complete
as possible.  If you have any other comments, I'd love to heart them.

A few things to keep in mind:
 There are several machines not listed that play the software of the
systems listed above (Sears Video Arcade, Franklin Ace, etc.)
 Many computer families (especially IBM) have had many upgrades
over the years, and not all games will be playable on all systems.
 A few non-American systems have not been listed because I know
shamefully little about them.  Sorry about that...
 Games for the NES, SNES, and Genesis systems are assumed to also be
available for their Japanese counterparts, the Famicom,
Super Famicom, and Mega Drive, respectively.  Therefore, these
latter systems will only be listed if the game is not available
for the non-Japanese system.
 The Neo*Geo is not listed, since the system uses the same hardware as
the arcade versions.  Neo*Geo games made for other systems are listed.

Following some of the games are a number:
(1) This is a non-arcade sequel to an arcade game
(2) This was originally a home game, later ported to the arcades
(3) This is not based on an arcade game, but contains
    similar elements (same characters, same movie tie-in, etc)

Following some of the system codes are a letter:
(a) This game is not released for this system under the same name.
    NOTE:  I do not include hacks or games that play in a similar way.
    This notation refers to a licensed game that, for whatever reason,
    had the name changed.  Port name follows wherever space allows.
(b) This game is or was offered free with this system.
(c) This game is (legally) available in packages with other games.
    NOTE:  This does not include those special/holiday packages
    offered occasionally by some companies.  This DOES include
    compilations that are put together under one name.  In the latter
    case, the compilation name follows.
(d) This is a CD game.
    NOTE: Some systems, such as the 3D0, only use CDs.  This notation
    would therefore be pointless, and is not used.

A (?) following a port name means will someone please confirm
the existence or non-existence of this port?

                      NOTES ON CROSS REFERENCES
Sequels that are known by a name other than "xxx II" etc. will
be cross-referenced from "xxx II" to the proper name.

In the case of arcade games (and their home ports) being released
under multiple names, all names will appear where the first
(alphabetically) is listed.  A cross reference will be placed
where all the other names appear.

Other references are used where necessary.
